## Estrutura da DRE

```
Receita Bruta
(-) Impostos (%)
= Receita Liquida
(-) Custos (Material, Manutencao)
= Lucro Bruto
(-) Desp. Operacionais (Aluguel, Salario, Contas, Internet, Outros)
= RESULTADO
```

## Lessons Learned

### Tecnico
- **Modificar arquivo unico** e rapido mas requer cuidado com a ordem das edicoes. Editar CSS, HTML e JS em blocos separados funcionou bem.
- **localStorage para config simples** e a escolha certa — evitou criar tabela no Supabase para 1 campo.
- **Mapeamento fixo por nome de categoria** e fragil (se renomear categoria no banco, quebra a DRE), mas aceitavel para MVP com categorias fixas.

### Processo
- **Ajuste visual pos-build** foi necessario. O primeiro render ficou funcional mas visualmente pobre. A iteracao rapida de CSS melhorou bastante.
- **Reutilizar funcoes existentes** (`getMesRange`, `formatMoney`) acelerou o desenvolvimento.

### O que Melhorar no Futuro
- Mapeamento categoria-grupo poderia vir do banco (coluna `grupo` na tabela categorias)
- Margem percentual em cada linha seria util para analise
- Comparacao mes anterior / ano anterior
